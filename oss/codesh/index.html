Codesh

A local-only macOS menu bar app that shows Codex session and weekly usage at a glance.

Codesh reads Codex session logs from disk and surfaces session/weekly usage percentages in the menu bar, so usage is visible without opening the CLI.

I built it as the new Codex app currently doesn't support this like the CodexMonitor app did. I wanted a tiny native utility focused on fast startup, local data only, and useful ambient feedback.
