React Native Paperplane

A tiny CLI that streamlines iOS TestFlight releases for React Native apps.

Paperplane handles the repetitive iOS release path: bumping build numbers, running build/export steps with Xcode, and uploading to TestFlight.

I built it to make release workflows deterministic, scriptable, and didn't want to use Xcode, EAS, or Fastlane etc. I just wanted something very simple and local.
